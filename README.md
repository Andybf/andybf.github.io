# PaintSpace
Uma tela em branco e várias opções de cores e formatos para você desenhar o que quiser!

# Sobre
PaintSpace é uma aplicação web que roda em página única (porém sem o uso de framework Angular) fortemente inspirada no Microsoft Paint. No PaintSpace podemos literalemnte desenhar o que quisermos, várias ferramentes, opções de formas e cores já vem disponíveis para você usar.

Linguagens Usadas:
* JavaScript - 75%
* CSS - 12,5%
* HTML - 12,5%

Frameworks/Bibliotecas Usadas:
* jQuery v3.3.1 (Já vem incluso) 
 
Suporte:
* Mozilla Firefox = 100%;
* Google Chrome = 100%;
* Safari = 100%
* É perfeitamente possível rodar em outros navegadores, porém não foram testados ainda, procure manter o seu navegador atualizado.
 
Execução:
* Simplesmente descompacte os arquivos em uma pasta de sua preferência, execute o arquivo `index.html` com o seu navegador preferido e aproveite.
